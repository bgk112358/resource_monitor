1. 工具清单

文件类型用途

2. Python 环境部署

2.1 依赖分析

两个 Python 脚本仅使用 Python 3 标准库，无第三方 pip 包依赖：

模块用途来源
sys, os, csv, subprocess, base64文件操作、进程调用、Base64 解码标准库，Python 3 自带
tkinterGUI 图形界面 (仅 plot_gui.py)标准库，Python 3 自带，但部分 Linux 发行版需单独安装 tk 组件

2.2 Ubuntu 22.04

# 1. 安装 Python 3 (默认已装)
sudo apt update

# 2. 验证

python3 -c "import tkinter; print('tk OK')"

# 1. 下载 Python 3.8+ 安装包

# 2. 安装时勾选 ✓ "Add Python to PATH"

#
# 3. 验证 (CMD 或 PowerShell)
python --version

3. app_profile — 设备端采集

3.1 部署

# 将 app_profile 拷贝到 TBox 设备的 /oem 目录
scp app_profile root@<TBox_IP>:/oem/
ssh root@<TBox_IP> chmod +x /oem/app_profile

3.2 用法

app_profile [-w <等待秒>] <PID | 程序路径> [duration_sec] [output_dir]

参数:
  -w N              启动后等待 N 秒再采样 (默认 3，路径模式有效)
  <PID | 程序路径>   目标进程 PID，或可执行程序路径（自动启动并监控）
  [duration_sec]     采集时长，秒 (默认 30)
  [output_dir]       输出目录 (默认 /oem/app_profile_<PID>_<时间戳>)

示例:
  /oem/app_profile 12345 60 /oem/my_profile        # 监控 PID=12345，60 秒
  /oem/app_profile /oem/tbox/bin/CloudGW 60        # 启动并监控 CloudGW
  /oem/app_profile -w 10 /oem/tbox/bin/HTCU 120    # 启动后等 10 秒，采集 120 秒

3.3 输出

文件内容
cpu.csv进程 CPU 占用 % (每秒)
mem.csvRSS / PSS / USS 内存 (KB)
threads_fd.csv线程数 + 文件描述符数
io.csv磁盘读/写吞吐 (KB/s)
core.csv每核心 CPU 占用 % (每秒)
net.csv各网卡上下行吞吐 (KB/s)
report.txt汇总报告 (均值/峰值等)

4. Python 可视化工具

4.1 plot_gui.py — GUI 桌面图表

python3 plot_gui.py <profile_dir>

# 示例
python3 plot_gui.py /oem/my_profile_12345_20260709_103238

# 快捷键: Ctrl-Q 或 Esc 退出

展示图表：CPU / Thread / FD 柱状图 + Memory / IO / Network / Per-Core 折线图。顶部显示版本匹配状态。

4.2 plot_profile.py — HTML 报告

python3 plot_profile.py <profile_dir>

# 示例
python3 plot_profile.py /oem/my_profile_12345_20260709_103238

# 输出: <profile_dir>/chart.html
# 浏览器打开即可

5. 注意事项

SELinux 设备：app_profile 启动时自动检测 SELinux 状态，尝试切换为 Permissive 模式以确保能读取 /proc/<PID>/io。如无法切换（需 root），会打印提示，IO 数据可能为 0。

6. 典型工作流

# === 设备端 (TBox) ===
# 1. 启动 app_profile 采集 60 秒
/oem/app_profile -w 5 /oem/tbox/bin/CloudGW 60 /oem/profile_test

# 2. 采集完成后，output 打好的 CSV 目录拷贝到 PC
scp -r root@<TBox_IP>:/oem/profile_test_* ~/Desktop/

# === 桌面端 (Ubuntu 或 Windows) ===
# 3. 生成 HTML 报告
python3 plot_profile.py ~/Desktop/profile_test_xxx

# 4. 浏览器打开 chart.html，或 GUI 查看
python3 plot_gui.py ~/Desktop/profile_test_xxx
